# Nur Athirah — Developer Portfolio

🌐 **Live site:** [nur-athirah-portfolio.vercel.app](https://nur-athirah-portfolio.vercel.app)

---

## Tech Stack

- **React** + **Vite**
- **CSS** (custom, no UI library)
- Deployed on **Vercel**

---

## Run Locally

```bash
# Clone the repo
git clone https://github.com/nurathirah29/nur-athirah-portfolio.git

# Navigate into the folder
cd my-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Then open `http://localhost:5173` in your browser.

---

## Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder.
